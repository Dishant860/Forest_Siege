## 🎮 About the Game

Forest Siege is a VR experience set inside a fragmented 
forest environment, designed to raise awareness about 
forest fragmentation through immersive gameplay.

Players explore the environment, interact with 
physics-based objects, and uncover an environmental 
narrative — all running on standalone Meta Quest 3 
hardware at a stable **72fps**.

---

## ✨ Key Features

- 🌲 Immersive forest environment with environmental storytelling
- ⚙️ Physics-based object interaction systems
- 🎯 Stable 72fps on standalone Meta Quest 3
- 🎤 Playtested with **8 real participants**
- 📊 Custom session data logging pipeline for research

---

## 🔬 Research Component

A custom **session data logging pipeline** records:

- Player position and movement data
- Object interaction events and timings
- Decision patterns across the environment

Data collected from **8 playtest participants** serves 
as a foundation for future **ML-based adaptive player 
behaviour analysis** using PyTorch.

## 🚀 Getting Started

### Prerequisites
- Unity 2022.3 LTS or later
- Meta Quest Developer Hub
- XR Interaction Toolkit package
- Meta XR SDK

### Setup
1. Clone the repository
2. Open in Unity Hub
3. Install required packages via Package Manager
4. Connect Meta Quest 3 via USB
5. Build and Run → Android → Meta Quest 3
